一：配置JDK

第一步：下载JDK

官网:https://www.oracle.com/cn/java/technologies/javase/javase-jdk8-downloads.html

用户注册登录之后下载安装完成。

第二步：验证JDK

输入 java -version 以及 javac -version查看java版本：这里一定要确保版本一致，不然会出现问题

之后输入 java -verbose查看jdk版本是否是对的：

我原来的版本是JRE，JRE虽然能运行JAVA文件，但是不是开发者模式，如果想要用Tomcat必须有JDK

JDK、JRE、JVM关系如下：

第三步：配置环境

1.JAVA_HOME 环境配置：C:\Program Files\Java\jdk1.8.0_261

2.Path 环境配置:在Path中添加：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

3.CLASSPATH环境配置： 添加： .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

配置完成!

二：配置Tomcat

第一步：下载Tomcat

Tomcat官网:https://tomcat.apache.org/download-80.cgi

推荐下载8.0版本，下载压缩包。

第二步：Tomcat环境变量配置

添加系统变量：

1:第一个环境变量：CATALINA_BASE

2：同理，第二个环境变量：CATALINA_HOME

3：在CLASSPATH环境下添加%CATALINA_HOME%\lib\servlet-api.jar;如果没有的话就新建，有的话就在原来的基础上添加。

4:在Path中加上Tomcat中的bin和lib文件即：%CATALINA_HOME%\bin;%CATALINA_HOME%\lib；

3、配置环境变量

检查发现MySQL安装在：

新增系统变量：MYSQL_HOME=C:\Program Files\MySQL\MySQL Server 5.7

在path变量下添加: %MYSQL_HOME%\bin
