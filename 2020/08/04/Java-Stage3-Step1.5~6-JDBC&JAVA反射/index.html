3-SQL注入漏洞

4-配置连接缓冲池

配置缓冲池能够有效的减少数据库的连接压力，从而更加有效。

如果没有配置连接缓冲池，则每个用户需要连接数据库时都需要访问连接数据库，造成数据库压力过大，如果配置了连接缓冲池，则每次只需要在缓冲池中获得连接，用完释放即可，不需要每次都连接数据库。

配置文件：c3p0-config.xml

配置完文件之后只需要获得连接池就可以直接使用 而不需要在设置参数。
