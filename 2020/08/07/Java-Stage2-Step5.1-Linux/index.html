2- linux命令

mkdir 创建文件夹

mkdir -p ： 创建多级文件夹

touch 创建文件

cp 复制目录文件

mv 移动目录或文件

rm 删除目录或文件 rm -rf

find / -name passwd 查找根目录下所有passwd，find最强大但是最慢

*任意字符任意个数 ？任意字符一个

useradd name 添加用户

useradd -g root wangwu 在root群组下创建wangwu用户

userdel name 删除用户

userdel -r name 删除用户以及用户的家目录

passwd wangwu 创建wangwu用户的密码

groupadd name 添加群组

groupdel name 删除群组

usermod -g g1 lisi 将lisi添加到群组g1

ll 查看权限

chmod 757 workspace01 修改workspace01的各个用户的权限 4-r 2-w 1-x

chown user002 1.txt 将1.txt文件的所有者改为user002

chgrp root 2.txt 将2.txt文件的所属组改为root
chown user002:user002 3.txt 将3.txt所有者 所属组改为user002

tar -cvf mytar.tar mydir 将mydir打包压缩成mytar.tar
