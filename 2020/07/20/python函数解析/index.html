Pytorch Tensor view clone

Tensor相当于numpy中的array

view保存的是远Tensor中的副本，也就是说原Tensor中的数值和新的数值是共享的，类似于指针的效果。

Pytorch index_select函数

第一个参数：对象

第二个参数：选择的维度，二维中，即行列

第三个参数：对应维度中选择的第几行等。

Pytorch random.shuffle

将list中的数随机排列

Pytorch gather

第一个参数：选择的维度

第二个参数：一个Tensor

返回的是y_hat中y对应的概率值

Pytorch cat

第一个参数：选择拼接的矩阵

第二参数：选择拼接的维度

Pytorch shape[-1]

获取最后一维的个数，如A.shape=(5,3),则shape[-1]=3

Pytorch torch.norm()求范数

判断类型是否相同

Python slice

第三个参数：跳数

Python *arg和**kwargs

*arg:任意个无名参数，类型是tuple

**kwargs:关键字参数，类型为dictionary 字典

Python 加2个维度的方法

Pytorch stack

第一个参数必须是tuple张量，不能是一维张量

Pytorch tensor.scatter_

第一个参数：用indices中的值替换的维度

第二个参数：替换维度的数

第三个参数：替换的数

例如：indices [0,1]=1 dim=0 则替换的位置为：[1,1],替换的数为x[0,1]=0.9476

Python contiguous()

如果在view之前使用了transpose,就需要用contiguous来吧tensor在内存中的分布变为连续分布的形式，不然无法使用view。
