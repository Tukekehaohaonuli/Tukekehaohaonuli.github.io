Restful

еӯҰд№ дҪҝз”ЁRestfulпјҢе№¶еңЁSpring bootдёӯе®һзҺ°гҖӮ

1-Spring Boot жһ„е»ә RESTful Web жңҚеҠЎ

Spring Boot йҖҡиҝҮдёҚеҗҢзҡ„жіЁи§ЈжқҘж”ҜжҢҒеүҚз«Ҝзҡ„иҜ·жұӮгҖӮ

  • @GetMappingпјҢеӨ„зҗҶ Get иҜ·жұӮ
  • @PostMappingпјҢеӨ„зҗҶ Post иҜ·жұӮ
  • @PutMappingпјҢз”ЁдәҺжӣҙж–°иө„жәҗ
  • @DeleteMappingпјҢеӨ„зҗҶеҲ йҷӨиҜ·жұӮ
  • @PatchMappingпјҢз”ЁдәҺжӣҙж–°йғЁеҲҶиө„жәҗ

жҲ‘д»¬дҪҝз”Ё ConcurrentHashMap жқҘжЁЎжӢҹеӯҳеӮЁ Message еҜ№иұЎзҡ„еўһеҲ ж”№жҹҘпјҢAtomicLong еҒҡдёәж¶ҲжҒҜзҡ„иҮӘеўһз»„е»әжқҘдҪҝз”ЁгҖӮConcurrentHashMap жҳҜ Java дёӯй«ҳжҖ§иғҪе№¶еҸ‘зҡ„ Map жҺҘеҸЈпјҢAtomicLong дҪңз”ЁжҳҜеҜ№й•ҝж•ҙеҪўиҝӣиЎҢеҺҹеӯҗж“ҚдҪңпјҢеҸҜд»ҘеңЁй«ҳе№¶еңәжҷҜдёӢиҺ·еҸ–еҲ°е”ҜдёҖзҡ„ Long еҖјгҖӮ

2-дҪҝз”Ё Swagger 2 жһ„е»ә RESTful APIs

Swagger жҳҜдёҖзі»еҲ— RESTful API зҡ„е·Ҙе…·пјҢйҖҡиҝҮ Swagger еҸҜд»ҘиҺ·еҫ—йЎ№зӣ®зҡ„дёҖз§ҚдәӨдә’ејҸж–ҮжЎЈпјҢе®ўжҲ·з«Ҝ SDK зҡ„иҮӘеҠЁз”ҹжҲҗзӯүеҠҹиғҪгҖӮдҪҝз”Ё Spring Boot йӣҶжҲҗ Swagger зҡ„зҗҶеҝөжҳҜпјҢдҪҝз”ЁжіЁи§ЈжқҘж Үи®°еҮәйңҖиҰҒеңЁ API ж–ҮжЎЈдёӯеұ•зӨәзҡ„дҝЎжҒҜпјҢSwagger дјҡж №жҚ®йЎ№зӣ®дёӯж Үи®°зҡ„жіЁи§ЈжқҘз”ҹжҲҗеҜ№еә”зҡ„ API ж–ҮжЎЈгҖӮSwagger иў«еҸ·з§°дё–з•ҢдёҠжңҖжөҒиЎҢзҡ„ API е·Ҙе…·пјҢе®ғжҸҗдҫӣдәҶ API з®ЎзҗҶзҡ„е…ЁеҘ—и§ЈеҶіж–№жЎҲпјҢ

ж·»еҠ дҫқиө–еҢ…

еҲӣе»ә SwaggerConfig й…ҚзҪ®зұ»

еңЁ SwaggerConfig зҡ„зұ»дёҠж·»еҠ дёӨдёӘжіЁи§Јпјҡ

  • @ConfigurationпјҢеҗҜеҠЁж—¶еҠ иҪҪжӯӨзұ»
  • @EnableSwagger2пјҢиЎЁзӨәжӯӨйЎ№зӣ®еҗҜз”Ё Swagger API ж–ҮжЎЈ

еңЁ SwaggerConfig дёӯж·»еҠ дёӨдёӘж–№жі•пјҡ

жӯӨж–№жі•дҪҝз”Ё @BeanпјҢеңЁеҗҜеҠЁж—¶еҲқе§ӢеҢ–пјҢиҝ”еӣһе®һдҫӢ DocketпјҲSwagger API ж‘ҳиҰҒпјүпјҢиҝҷйҮҢйңҖиҰҒжіЁж„Ҹзҡ„жҳҜ .apis(RequestHandlerSelectors.basePackage(вҖңcom.neo.xxxвҖқ)) жҢҮе®ҡйңҖиҰҒжү«жҸҸзҡ„еҢ…и·Ҝеҫ„пјҢеҸӘжңүжӯӨи·Ҝеҫ„дёӢзҡ„ Controller зұ»жүҚдјҡиҮӘеҠЁз”ҹжҲҗ Swagger API ж–ҮжЎЈгҖӮ

иҝҷеқ—й…ҚзҪ®зӣёеҜ№йҮҚиҰҒдёҖдәӣпјҢдё»иҰҒй…ҚзҪ®йЎөйқўеұ•зӨәзҡ„еҹәжң¬дҝЎжҒҜеҢ…жӢ¬пјҢж ҮйўҳгҖҒжҸҸиҝ°гҖҒзүҲжң¬гҖҒжңҚеҠЎжқЎж¬ҫгҖҒиҒ”зі»ж–№ејҸзӯүпјҢжҹҘзңӢ ApiInfo зұ»зҡ„жәҗз ҒиҝҳдјҡеҸ‘зҺ°ж”ҜжҢҒ license й…ҚзҪ®зӯүгҖӮ

й…ҚзҪ®е®ҢжҲҗд№ӢеҗҺеҗҜеҠЁйЎ№зӣ®пјҢеңЁжөҸи§ҲеҷЁдёӯиҫ“е…ҘзҪ‘еқҖ http://localhost:8080/swagger-ui.htmlпјҢеҚіеҸҜзңӢеҲ°дёҠйқўзҡ„й…ҚзҪ®дҝЎжҒҜ

Swagger еёёз”ЁжіЁи§Ј

Swagger йҖҡиҝҮжіЁи§ЈиЎЁжҳҺиҜҘжҺҘеҸЈдјҡз”ҹжҲҗж–ҮжЎЈпјҢеҢ…жӢ¬жҺҘеҸЈеҗҚгҖҒиҜ·жұӮж–№жі•гҖҒеҸӮж•°гҖҒиҝ”еӣһдҝЎжҒҜзӯүпјҢеёёз”ЁжіЁи§ЈеҶ…е®№еҰӮдёӢпјҡ

дҪңз”ЁиҢғеӣҙ API дҪҝз”ЁдҪҚзҪ®
еҚҸи®®йӣҶжҸҸиҝ° @Api з”ЁдәҺ Controller зұ»дёҠ
еҚҸи®®жҸҸиҝ° @ApiOperation з”ЁеңЁ Controller зҡ„ж–№жі•дёҠ
йқһеҜ№иұЎеҸӮж•°йӣҶ @ApiImplicitParams з”ЁеңЁ Controller зҡ„ж–№жі•дёҠ
йқһеҜ№иұЎеҸӮж•°жҸҸиҝ° @ApiImplicitParam з”ЁеңЁ @ApiImplicitParams зҡ„ж–№жі•йҮҢиҫ№
е“Қеә”йӣҶ @ApiResponses з”ЁеңЁ Controller зҡ„ж–№жі•дёҠ
е“Қеә”дҝЎжҒҜеҸӮж•° @ApiResponse з”ЁеңЁ @ApiResponses йҮҢиҫ№
жҸҸиҝ°иҝ”еӣһеҜ№иұЎзҡ„ж„Ҹд№ү @ApiModel з”ЁеңЁиҝ”еӣһеҜ№иұЎзұ»дёҠ
еҜ№иұЎеұһжҖ§ @ApiModelProperty з”ЁеңЁеҮәе…ҘеҸӮж•°еҜ№иұЎзҡ„еӯ—ж®өдёҠ

@Api зҡ„дҪҝз”Ё

Api дҪңз”ЁеңЁ Controller зұ»дёҠпјҢеҒҡдёә Swagger ж–ҮжЎЈиө„жәҗпјҢиҜҘжіЁи§Је°ҶдёҖдёӘ ControllerпјҲClassпјүж ҮжіЁдёәдёҖдёӘ Swagger иө„жәҗпјҲAPIпјүгҖӮеңЁй»ҳи®Өжғ…еҶөдёӢпјҢSwagger-Core еҸӘдјҡжү«жҸҸи§Јжһҗе…·жңү @Api жіЁи§Јзҡ„зұ»пјҢиҖҢдјҡиҮӘеҠЁеҝҪз•Ҙе…¶д»–зұ»еҲ«иө„жәҗпјҲJAX-RS endpointsгҖҒServlets зӯүпјүзҡ„жіЁи§ЈгҖӮ

дҪҝз”ЁзӨәдҫӢпјҡ

еұһжҖ§еҗҚз§° еӨҮжіЁ
value url зҡ„и·Ҝеҫ„еҖј
tags еҰӮжһңи®ҫзҪ®иҝҷдёӘеҖјпјҢvalue зҡ„еҖјдјҡиў«иҰҶзӣ–
description еҜ№ API иө„жәҗзҡ„жҸҸиҝ°
produces For example, вҖңapplication/json, application/xmlвҖқ
consumes For example, вҖңapplication/json, application/xmlвҖқ
protocols Possible values: http, https, ws, wss
authorizations й«ҳзә§зү№жҖ§и®ӨиҜҒж—¶й…ҚзҪ®
hidden й…ҚзҪ®дёә true е°ҶеңЁж–ҮжЎЈдёӯйҡҗи—Ҹ

иҮӘеҠЁе°Ҷ MessageController еҶ…зҡ„ж–№жі•йғҪж·»еҠ дәҶжҳ е°„пјҢе№¶ж ҮжҳҺдәҶжҜҸз§Қж–№жі•зҡ„иҜ·жұӮж–№ејҸгҖӮ

@ApiOperation зҡ„дҪҝз”Ё

ApiOperation е®ҡд№үеңЁж–№жі•дёҠпјҢжҸҸиҝ°ж–№жі•еҗҚгҖҒж–№жі•и§ЈйҮҠгҖҒиҝ”еӣһдҝЎжҒҜгҖҒж Үи®°зӯүдҝЎжҒҜгҖӮ

дҪҝз”ЁзӨәдҫӢпјҡ

еұһжҖ§еҗҚз§° еӨҮжіЁ
value url зҡ„и·Ҝеҫ„еҖј
tags еҰӮжһңи®ҫзҪ®иҝҷдёӘеҖјгҖҒvalueзҡ„ еҖјдјҡиў«иҰҶзӣ–
produces For example, вҖңapplication/json, application/xmlвҖқ
consumes For example, вҖңapplication/json, application/xmlвҖқ
protocols Possible values: http, https, ws, wss
authorizations й«ҳзә§зү№жҖ§и®ӨиҜҒж—¶й…ҚзҪ®
hidden й…ҚзҪ®дёә true е°ҶеңЁж–ҮжЎЈдёӯйҡҗи—Ҹ
response иҝ”еӣһзҡ„еҜ№иұЎ
responseContainer иҝҷдәӣеҜ№иұЎжҳҜжңүж•Ҳзҡ„ вҖңListвҖқ, вҖңSetвҖқ or вҖңMapвҖқпјҢе…¶д»–ж— ж•Ҳ
httpMethod вҖңGETвҖқгҖҒвҖқHEADвҖқгҖҒвҖқPOSTвҖқгҖҒвҖқPUTвҖқгҖҒвҖқDELETEвҖқгҖҒвҖқOPTIONSвҖқ and вҖңPATCHвҖқ
code http зҡ„зҠ¶жҖҒз Ғ й»ҳи®Ө 200
extensions жү©еұ•еұһжҖ§

йҮҚеҗҜйЎ№зӣ®д№ӢеҗҺпјҢеңЁжөҸи§ҲеҷЁдёӯиҫ“е…ҘзҪ‘еқҖ http://localhost:8080/swagger-ui.html#/message-controller/listUsingGETпјҢеҸҜд»ҘзңӢеҲ°еҰӮдёӢж•Ҳжһңпјҡ

@ApiImplicitParams е’Ң @ApiImplicitParam зҡ„дҪҝз”Ё

@ApiImplicitParams з”ЁдәҺжҸҸиҝ°ж–№жі•зҡ„иҝ”еӣһдҝЎжҒҜпјҢе’Ң @ApiImplicitParam жіЁи§Јй…ҚеҗҲдҪҝз”Ёпјӣ@ApiImplicitParam з”ЁжқҘжҸҸиҝ°е…·дҪ“жҹҗдёҖдёӘеҸӮж•°зҡ„дҝЎжҒҜпјҢеҢ…жӢ¬еҸӮж•°зҡ„еҗҚз§°гҖҒзұ»еһӢгҖҒйҷҗеҲ¶зӯүдҝЎжҒҜгҖӮ

дҪҝз”ЁзӨәдҫӢпјҡ

еұһжҖ§еҗҚз§° еӨҮжіЁ
name жҺҘж”¶еҸӮж•°еҗҚ
value жҺҘж”¶еҸӮж•°зҡ„ж„Ҹд№үжҸҸиҝ°
required еҸӮж•°жҳҜеҗҰеҝ…еЎ«еҖјдёә true жҲ–иҖ… false
dataType еҸӮж•°зҡ„ж•°жҚ®зұ»еһӢеҸӘдҪңдёәж Үеҝ—иҜҙжҳҺпјҢе№¶жІЎжңүе®һйҷ…йӘҢиҜҒ
paramType жҹҘиҜўеҸӮж•°зұ»еһӢпјҢе…¶еҖјпјҡ path д»Ҙең°еқҖзҡ„еҪўејҸжҸҗдәӨж•°жҚ® query зӣҙжҺҘи·ҹеҸӮж•°е®ҢжҲҗиҮӘеҠЁжҳ е°„иөӢ body д»ҘжөҒзҡ„еҪўејҸжҸҗдәӨпјҢд»…ж”ҜжҢҒ POST header еҸӮж•°еңЁ request headers йҮҢиҫ№жҸҗдәӨ form д»Ҙ form иЎЁеҚ•зҡ„еҪўејҸжҸҗдәӨ д»…ж”ҜжҢҒ POST
defaultValue й»ҳи®ӨеҖј

еңЁжөҸи§ҲеҷЁдёӯиҫ“е…ҘзҪ‘еқҖ http://localhost:8080/swagger-ui.html#/message-controller/createUsingPOST

@ApiResponses е’Ң @ApiResponse зҡ„дҪҝз”Ё

@ApiResponses дё»иҰҒе°ҒиЈ…ж–№жі•зҡ„иҝ”еӣһдҝЎжҒҜе’Ң @ApiResponse й…ҚзҪ®иө·жқҘдҪҝз”ЁпјҢ@ApiResponse е®ҡд№үиҝ”еӣһзҡ„е…·дҪ“дҝЎжҒҜеҢ…жӢ¬иҝ”еӣһз ҒгҖҒиҝ”еӣһдҝЎжҒҜзӯүгҖӮ

дҪҝз”ЁзӨәдҫӢпјҡ

еұһжҖ§еҗҚз§° еӨҮжіЁ
code http зҡ„зҠ¶жҖҒз Ғ
message жҸҸиҝ°
response й»ҳи®Өе“Қеә”зұ» Void
reference еҸӮиҖғ
responseHeaders е°ҒиЈ…иҝ”еӣһдҝЎжҒҜ
responseContainer еӯ—з¬ҰдёІ

еңЁжөҸи§ҲеҷЁдёӯиҫ“е…ҘзҪ‘еқҖ http://localhost:8080/swagger-ui.html#/message-controller/modifyUsingPUT

@ApiModel е’Ң @ApiModelProperty зҡ„дҪҝз”Ё

еңЁе®һйҷ…зҡ„йЎ№зӣ®дёӯжҲ‘д»¬еёёеёёдјҡе°ҒиЈ…дёҖдёӘеҜ№иұЎдҪңдёәиҝ”еӣһеҖјпјҢ@ApiModel е°ұжҳҜиҙҹиҙЈжҸҸиҝ°еҜ№иұЎзҡ„дҝЎжҒҜпјҢ@ApiModelProperty иҙҹиҙЈжҸҸиҝ°еҜ№иұЎдёӯеұһжҖ§зҡ„зӣёе…іеҶ…е®№гҖӮ

дҪҝз”ЁзӨәдҫӢпјҡ

еұһжҖ§й…ҚзҪ®еҰӮдёӢиЎЁжүҖзӨәпјҡ

еұһжҖ§еҗҚз§° еӨҮжіЁ
value еұһжҖ§жҸҸиҝ°
name еҰӮжһңй…ҚзҪ®иҰҶзӣ–еұһжҖ§еҗҚз§°
allowableValues е…Ғи®ёзҡ„еҖј
access еҸҜд»ҘдёҚй…ҚзҪ®
notes жІЎжңүдҪҝз”Ё
dataType ж•°жҚ®зұ»еһӢ
required жҳҜеҗҰдёәеҝ…дј еҸӮж•°
position жҳҫзӨәзҡ„йЎәеәҸдҪҚзҪ®
hidden жҳҜеҗҰеӣ жӯӨ
example дёҫдҫӢ
readOnly еҸӘиҜ»
reference еј•з”Ё

иҝҷж ·жҲ‘д»¬еңЁ Controller дёӯе°ҒиЈ…иҝ”иҝҳдҝЎжҒҜж—¶е°ұеҸҜд»Ҙиҝҷж ·ж“ҚдҪңпјҡ

еңЁжөҸи§ҲеҷЁдёӯиҫ“е…ҘзҪ‘еқҖ http://localhost:8080/swagger-ui.htmlпјҢзӮ№и§ЈйЎөйқў Models жҠҳеҸ йЎ№

зҒөжҙ»ең°еҲ©з”ЁиҝҷдәӣжіЁи§Је°ұеҸҜд»ҘиҮӘеҠЁжһ„е»әеҮәжё…жҷ°зҡ„ API ж–ҮжЎЈгҖӮ

Try it out

дҪҝз”Ё Swagger еҲӣе»әзҡ„еңЁзәҝ API иҝҳжңүдёҖдёӘйқһеёёејәеӨ§зҡ„еҠҹиғҪпјҢеҸҜд»ҘеңЁйЎөйқўзӣҙжҺҘжөӢиҜ•жҺҘеҸЈзҡ„еҸҜз”ЁжҖ§пјҢиҝҷж ·еңЁеүҚз«Ҝе’ҢеҗҺз«ҜжҺҘеҸЈи°ғиҜ•еҮәзҺ°й—®йўҳж—¶пјҢеҸҜд»Ҙйқһеёёж–№дҫҝең°еҲ©з”ЁжӯӨеҠҹиғҪиҝӣиЎҢжҺҘеҸЈйӘҢиҜҒгҖӮеңЁдёҠйқўеҸӮж•°и®Іи§ЈиҝҮзЁӢдёӯпјҢжҲ‘д»¬еҸ‘зҺ°жҜҸдёӘжҺҘеҸЈжҸҸиҝ°еҸідҫ§йғҪжңүдёҖдёӘжҢүй’® try it outпјҢеҚ•еҮ» try it out жҢүй’®еҚіеҸҜиҝӣе…ҘиЎЁеҚ•йЎөйқў

еңЁиЎЁеҚ•йЎөйқўж·»еҠ зӣёе…іеӯ—ж®өеҗҺпјҢеҚ•еҮ»вҖңExecuteвҖқжҢүй’®е°ұдјҡе°ҶиҜ·жұӮеҸ‘йҖҒеҲ°еҗҺеҸ°пјҢд»ҺиҖҢиҝӣиЎҢжҺҘеҸЈйӘҢиҜҒгҖӮ

еңЁеҗҺз«Ҝи°ғж•ҙ Swagger ж–№жі•дёҠеҜ№еә”еҸӮж•°пјҢеҚіеҸҜзңӢеҲ° curl е‘Ҫд»ӨеҸӮж•°зҡ„еҸҳеҢ–гҖӮ
